HH.ru Automation Panel

Dependencies

Install system dependencies for Tkinter:

Ubuntu / Debian Fedora / RHEL Arch / Manjaro 
# Ubuntu / Debian
sudo apt update && sudo apt install python3-tk python3-dev

# Fedora / RHEL / CentOS
sudo dnf install python3-tkinter python3-devel

# Arch Linux / Manjaro
sudo pacman -S tk

Installation

Install Python dependencies:

pip install -r requirements.txt

Configuration

The application will automatically generate:

hh_cookies.json (after browser authorization)
.env with N8N_RUNNERS_AUTH_TOKEN

Run

python app.py

Docker Requirement

Docker and Docker Compose must be installed to run n8n containers. 
sudo systemctl start docker

Stop

Close the application window. All Docker containers and background processes will terminate automatically.
